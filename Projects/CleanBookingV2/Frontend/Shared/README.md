# Shared Frontend

Placeholder for code that may eventually be reused by both frontend apps.

Do not move code here just because it could be shared. Move code here when both
`Public` and `Private` actually need it.

Likely future candidates:

- API clients
- date and time helpers
- validation helpers
- shared UI components such as status messages or calendars

Keeping this folder empty for now avoids guessing too early while still showing
where shared frontend code should live later.
